# TabBar 图标说明

由于微信小程序 tabBar 需要本地图片文件，请在此目录下放置以下图标文件：

## 需要的图标文件：

1. **home.png** / **home-active.png** - 礼品管理图标
2. **exchange.png** / **exchange-active.png** - 兑换管理图标  
3. **points.png** / **points-active.png** - 积分管理图标
4. **profile.png** / **profile-active.png** - 用户中心图标

## 图标规格要求：

- 尺寸：81px × 81px（推荐）
- 格式：PNG（支持透明背景）
- 颜色：未选中状态使用灰色，选中状态使用主题绿色 (#07c160)

## 临时方案：

如果暂时没有图标文件，可以：
1. 使用在线图标生成工具创建
2. 使用设计工具（如 Figma、Sketch）导出
3. 使用简单的文字图标或emoji（需要转换为图片）

## 图标建议：

- 礼品：📦 或购物袋图标
- 兑换：🎫 或票据图标
- 积分：⭐ 或星星图标
- 我的：👤 或用户图标
